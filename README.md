# AnnaSetu

**Rescuing Food, Serving Communities**

AnnaSetu is a full-stack web platform designed to reduce food waste by connecting food donors such as restaurants, hostels, bakeries, and event organizers with NGOs and community receivers who can redistribute surplus food to people in need.

The platform enables donors to post available food, allows receivers to browse and request donations, and provides a structured workflow for managing pickup and delivery before the food expires.

## Features
- Role-based login for donors, NGOs, and admin
- Create and manage food donation listings
- Browse and filter available food donations
- Request and accept donation pickups
- Track donation status from availability to delivery
- Dashboard for monitoring donation activity and impact

## Tech Stack
- Frontend: React
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## Goal
To build a practical web solution that helps reduce food wastage and improve food redistribution in communities.
